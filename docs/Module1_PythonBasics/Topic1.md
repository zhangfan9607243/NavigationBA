# Topic 1 - Python简介

## 1. Python语言简介

### 1.1. Python 的起源与发展历程

- Python语言是1989年由Guido van Rossum发明的，灵感来源于教学语言ABC。ABC是由吉多参加设计的一种教学语言，ABC这种语言非常优美和强大，是**专门为非程序员设计的**。但是ABC语言并没有成功，吉多认为原因是非开放造成的。因此吉多决心设计一种新的编程语言，继承ABC优雅简单的特点，但是要保持开放。
- Python的名字取自英国喜剧节目《蒙提·派森的飞行马戏团》（Monty Python’s Flying Circus），而非“蟒蛇”，虽然之后Python的官方logo就是蟒蛇。
- Python语言的第一个完整版由Guido于1991年发布，并且开源，全世界工程师都可以参与到Python的开发与改进中。此后，Python凭借简洁语法和强大功能，逐渐在学术界与工业界流行。
- 在经历近10年的发展完善后，Python 2版本于2000年发布，Python 2就已经是当时应用最广泛的计算机语言。
- Python 3版本于2008年发布，并且沿用至今，截至2016年，Python 3版本及重要的第三方库已基本完善。
- 截至目前，Python最新版已经是3.14，Python 2版本早已停止更新但仍可使用，Python 1版本已经在现代操作系统中不能使用。
- 如今，Python已成为全球最受欢迎的编程语言之一，在数据分析、人工智能、Web开发等领域应用广泛

### 1.2. Python 的语言特点

- **简洁易学**：语法接近自然语言，代码量少，开发效率高。
- **设计哲学**：强调优雅、明确、简单，主张“用一种方法来完成一件事”。
- **生态丰富**：Python有着海量的标准库与第三方库，涵盖数据处理、网络通信、人工智能、科学计算等各类应用场景。
- **跨平台性**：支持多种操作系统，几乎可以在任何环境中运行。
- **可扩展性**：能与 C/C++ 等语言结合，兼顾灵活性与性能。
- **社区活跃**：使用广泛，拥有庞大的开发者社区。

## 2. Python与PyCharm安装
### 2.1 Python解释器安装
Python是一门解释型语言，运行程序需要依赖**解释器**。我们常说的安装Python，实际上就是在安装Python的解释器。

Python解释器就好比是一个字典，计算机的出厂设置是只听得懂机器语言（0 和 1），要想听懂其他的编程语言，如Python、C、Java等，就必须依赖解释器来将编程语言翻译成机器语言。解释器的作用就是在Python语言和计算机语言之间做实时翻译：当你写下一行 Python 代码时，解释器会把它一句一句翻译成机器能理解的指令，然后交给计算机执行。因此，没有解释器，计算机是无法直接运行Python程序的。

Python解释器官方下载地址：[https://www.python.org](https://www.python.org)

- 建议安装Python 3.9之后的任意的Python 3.x版本，避免使用老旧的Python 2.x版本；
- 要选择与自己操作系统相匹配的安装器；
- Python解释器安装到电脑上之后，就是以一个应用程序的形式存在的，并且它所在的路径（文件夹）很重要；
- Python解释器安装完毕后，可以在终端运行以下命令检查是否安装成功：  

```bash
python --version
```

### 2.2 PyCharm集成开发环境安装
安装完 Python 解释器之后，我们还需要使用集成开发环境（IDE）来编程效率。虽然我们已经有了一个 Python 的字典让计算机理解Python语言，但是翻阅一个字典其实远不如翻译软件效率高，集成开发环境（IDE）就相当于是一个翻译软件。

Python 解释器就像一本厚重的字典，可以让计算机逐条理解 Python 语言，但仅靠查阅字典往往既慢又不方便。而 IDE 则更像是一款智能翻译软件，不仅能帮电脑快速翻译，还提供了许多便捷的编程功能，大大提升了开发体验和效率。

PyCharm 是目前应用最广泛的 Python IDE之一。虽然许多人在学习后期可能会转向更为通用和强大的开发工具，如 VSCode，但 VSCode 面向所有编程语言，功能丰富的同时也增加了上手难度。相比之下，PyCharm 专为 Python 开发而设计，界面简洁、功能集中，更适合初学者快速入门。掌握 PyCharm 后，再过渡到 VSCode 等更高级的 IDE 会更加顺利。

PyCharm 官方下载地址：[https://www.jetbrains.com/pycharm](https://www.jetbrains.com/pycharm)

- PyCharm 分为社区版（免费）和专业版（收费），学习和日常使用安装社区版就足够。专业版虽然收费，但是在校学生只要提供学校邮箱或上传在读证明的照片，就可以免费使用。
- 安装完成后，首次启动需要设置：
    - 配置 Python 解释器路径；
    - 新建或导入项目；
    - 新建 Python 文件；
    - 运行一个简单的程序来测试配置。

### 2.3 其他运行Python的方式（补充）
除了在 PyCharm 中运行 Python 程序外，我们还可以通过多种方式来体验和使用 Python：

#### (1) Python 命令运行 Python 文件：
- 在编写好 Python 文件后，可以在终端命令行中输入 `python + 文件名.py` 的命令来运行该文件；
- 这种方法是Python程序运行的最基本方法，简单直接，执行时会从头到尾运行整个文件，但是不便于调试。

#### (2) Python 交互式命令行：  
- 安装 Python 后，可以在终端命令行中输入 `python` 进入交互式环境，运行结束后使用 `exit()` 退出；
- 这种方法简单直接，适合快速测试语句或函数，但不适合编写较长程序。

#### (3) IPython  
- IPython 是对标准 Python 交互式环境的增强版；
- IPython 需要提前安装，安装命令是 `pip install ipython`；
- 之后在终端命令行中输入 `ipython` 进入 IPython 的交互式环境，运行结束后使用 `quit()` 退出。

#### (4) Jupyter Notebook
- Jupyter Notebook 是基于网页的交互式开发环境，可以将代码、运行结果、图表、文字说明融合在一个文档中；
- 它最大的特点是支持分步运行，适合数据分析与可视化，支持边写代码边记录思路，并且可导出为 HTML、PDF 等格式，方便分享；
- Jupyter Notebook 中运行 Python 的并不是 `.py` 格式的文件，而是 `.ipynb` 格式的文件；
- Jupyter Notebook 的安装需要在终端中输入 `pip install notebook` 命令，安装好之后需要使用 `jupyter notebook` 命令运行，运行之后会在本地计算机的浏览器上开启一个网页，所有代码编辑和运行都是在网页里进行的。

## 3. Python学习技巧
学习编程并不是一蹴而就的过程，而是一段不断尝试、不断修正、不断积累的旅程。只要保持好奇心与耐心，多练习、多思考，并在实践中收获成就感与自信心。  

- **动手实践**：一定要多加练习，边学边写，只有通过自己动手构建代码，才能真正掌握知识。  
- **理清逻辑**：在编程之前先理清思路，写好算法步骤，再动手实现代码，避免盲目下手。  
- **循序渐进**：把问题拆解为小步骤，逐步探索和验证，稳扎稳打地提升能力。  
- **善待错误**：优秀的程序员面对错误和 Bug 时，总是保持好奇心，把它们当作学习和进步的机会，而不是烦躁的来源。  
- **善用工具**：Google 上可以找到许多和你遇到相同问题的解决方案；ChatGPT 也可以作为很好的学习助手，但在初学阶段要避免过度依赖，确保自己理解与思考。  
- **重用资源**：充分利用成熟的库、速查表，以及自己已经验证过的代码片段，提高效率，避免重复造轮子。  
- **注重实用**：学习编程的价值不在于炫技，而在于是否能真正解决实际问题、为决策提供有用的信息。
